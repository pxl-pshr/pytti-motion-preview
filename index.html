Real-time camera path visualizer for AI video

A browser-based tool for previewing PYTTI camera motion expressions before committing to a full render. Paste your translate/rotate expressions, scrub the timeline, and see the 3D path and live camera view update instantly. No installation. No server. Open the file and go.

Launch Tool PyTTI Docs
01
Expression Engine
Type any math expression using t as the frame variable. sin, cos, radians, degrees — all the PYTTI-compatible functions are available.
02
3D Path View
Watch the accumulated camera trajectory render live in a 3D observer scene. Gradient line from start to end, with tick marks and frustum at the current frame.
03
Camera View
Switch to first-person camera view to see exactly what the AI diffusion model's virtual camera sees at each frame of the animation.
04
Quaternion Engine
Rotation uses PYTTI's native [w, x, y, z] quaternion format. Accumulated quaternion and Euler readouts update per-frame so you can debug rotation drift.
05
Path Analysis
Automatic loop detection, total path distance, and max radius from origin — quickly validate whether your motion will return to start.
06
Zero Dependencies
Single HTML file. Three.js loaded from CDN. Open directly from disk in any modern browser. No build step, no npm, no server.
Example Expressions
# Gentle orbit translate_x: 20 * cos(0.2 * t) translate_y: 10 * sin(0.2 * t) translate_z: 0 rotate_3d: [cos(radians(0.5)), 0, sin(radians(0.5)), 0] # Slow zoom with yaw translate_x: 0 translate_y: 0 translate_z: -3 rotate_3d: [cos(radians(1)), 0, sin(radians(1)), 0] # Spiral descent translate_x: sin(0.1 * t) * 5 translate_y: -1 translate_z: cos(0.1 * t) * 5 rotate_3d: [1, 0, 0, 0]
// paste expressions directly from your pytti config
How To Use
01
Open the tool
Click "Launch Tool" above, or open pytti-motion-preview.html directly in your browser from disk.
02
Paste your expressions
Enter translate_x/y/z and rotate_3d expressions in the left panel. The path recomputes automatically as you type (400ms debounce).
03
Scrub or play
Use the timeline slider or hit Play to animate. Adjust frame count and FPS to match your intended render settings.
04
Switch views
Toggle between PATH (3D observer) and CAM (first-person) to validate both the trajectory shape and the frame-by-frame camera perspective.
05
Confirm and render
When the path looks right, copy the expressions back into your pytti config. What you see here is what PyTTI will compute.
